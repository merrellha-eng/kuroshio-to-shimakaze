# TASK.md｜黒潮と島風 公式ホームページ制作

## 1. プロジェクト概要

- サイト名：黒潮と島風
- 業種：寿司・創作料理店
- 制作対象：店舗ホームページ（1ページ構成）
- 目的：店舗コンセプト、料理、店内の特徴、アクセス、予約導線を分かりやすく伝える

## 2. 作業対象フォルダ

```text
codex-kit/
├─ AGENTS.md
└─ projects/
   └─ kuroshio-to-shimakaze/
      ├─ TASK.md
      ├─ reference/
      │  └─ reference-lp.png
      └─ assets/
         └─ images/
```

- 参考LP画像：`projects/kuroshio-to-shimakaze/reference/reference-lp.png`
- ホームページで使用する単体素材画像：`projects/kuroshio-to-shimakaze/assets/images/`
- 共通ルールは `codex-kit/AGENTS.md` を確認すること

## 3. 実装方針

- `reference/reference-lp.png` を確認し、全体構成・デザイン・配色・余白・画像配置・装飾・文字の雰囲気を可能な限り忠実に再現する
- 実装で使用する画像は `assets/images/` 内の素材を使用する
- 参考LP画像そのものをページ全体の画像として表示する実装は禁止
- LP内の画像部分を参考画像から切り抜いて素材として流用しない
- 素材画像が不足している箇所は、既存素材で不自然な代替を行わず、実装完了時に不足素材として報告する
- 文言は参考LP画像に掲載されている内容を基本として実装する
- ナビゲーション、ボタン、メニューカード、予約・問い合わせ導線などはWebページとして操作可能な要素で実装する

## 4. 必要セクション

参考LP画像を基準として、少なくとも以下の内容を実装する。

- ヘッダー／ナビゲーション
- ファーストビュー
- コンセプト紹介
- おすすめメニュー
- アクアポニックス・新鮮野菜の紹介
- 店内・カウンター空間の紹介
- 本日のおすすめ
- SNS案内
- アクセス／店舗情報
- 予約・問い合わせ導線
- フッター

## 5. 画像・アイコンの取り扱い

- 写真素材は `assets/images/` に保存されたファイルを使用する
- ロゴ画像が用意されている場合は `assets/images/` 内のロゴ素材を使用する
- アイコンが必要な場合は Google Fonts Icons（Material Symbols）を使用してよい
- 画像には用途に応じた `alt` を設定する
- 画像表示でレイアウト崩れが起きないよう、適切に `object-fit` 等を設定する

## 6. 実装要件

- PC表示では参考LP画像の印象に忠実なレイアウトとする
- スマートフォン表示に対応し、読みやすさと操作性を維持する
- ナビゲーションはスマートフォン表示で適切に切り替える
- ボタンやリンクにはホバー・フォーカス時の視認性を確保する
- セマンティックなHTML構造を使用する
- CSS・JavaScriptは整理し、不要なライブラリは追加しない
- 外部リンクや未確定の予約機能はダミーリンクとして扱い、誤送信・誤遷移を起こさないようにする

## 7. ページ内リンク・導線

- ヘッダーナビゲーションは該当セクションへのページ内リンクとして機能させる
- メニュー一覧、アクセス、予約・問い合わせ等のボタンは視覚的に再現する
- 予約フォームや外部予約サービスは今回実装対象外とし、ボタンはダミーリンクまたは未接続の状態で実装する

## 8. 確認項目・完了条件

- 参考LP画像を確認しながら、PC表示で全体の見た目を忠実に再現できている
- スマートフォン表示でもレイアウト崩れがない
- 画像パス、リンク、ページ内リンクにエラーがない
- 使用素材は `assets/images/` から参照されている
- 参考LP画像を完成ページの代替として貼り付けていない
- 不足している画像素材や未接続の機能がある場合は、完了報告時に一覧で伝える
- 最後に変更したファイルと確認結果を簡潔に報告する
